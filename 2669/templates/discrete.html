跨区域经营风险-收益评估

以风险等级 R 与收益等级 E 为坐标，联动项目级、城市级、单位级三层颗粒度，形成可筛选、可下钻、可落地的经营决策视图。

维度指标规则 R 风险等级：区域合规 1.0 + 合同底线 1.0 + 客户健康 0.8 + 资金安全 0.8 + 履约真实 0.4，加权后离散为 1/2/3 档。 E 收益等级：盈利水平 1.05 + 产值转化 0.75 + 资金回收 0.75 + 合同规模 0.45，加权后离散为 1/2/3 档。 分箱阈值：得分 <= 1.6 为低，1.6 < 得分 <= 2.4 为中，得分 > 2.4 为高。 坐标解释：横轴为风险等级 R，纵轴为收益等级 E；背景色块与右侧九宫格策略区一一对应，点击散点、矩阵或地图可联动筛选。
维度指标规则 风险等级 R 和收益等级 E 均被离散化为 1（低）、2（中）、3（高）三个级别。

- 风险等级 R：综合考量“区域合规”(权重1.0)、“合同底线”(权重1.0)、“客户健康”(权重0.8)、“资金安全”(权重0.8)和“履约真实”(权重0.4)共五个子维度，加权汇总后进行分箱：R得分 ≤ 1.6 为低风险，1.6 < R得分 ≤ 2.4 为中风险，R得分 > 2.4 为高风险。
- 收益等级 E：综合考量“盈利水平”(权重1.05)、“产值转化”(权重0.75)、“资金回收”(权重0.75)和“合同规模”(权重0.45)共四个子维度，加权汇总后进行分箱：E得分 ≤ 1.6 为低收益，1.6 < E得分 ≤ 2.4 为中收益，E得分 > 2.4 为高收益。

散点图的横轴表示风险等级 R，纵轴表示收益等级 E。 散点图中的每个气泡代表一个项目（或城市/单位），其位置由其 R 和 E 离散等级决定。
散点图的背景色块区域与右侧九宫格策略分区（例如“淘汰区”、“扩张区”）严格一一对应。
您可以通过点击散点图中的气泡、九宫格中的区域、地图上的省份/城市或下方的 Tab 切换按钮来同步筛选详细数据。
