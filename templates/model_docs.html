怎么算分？—— 三种逻辑，一句话讲清

红线类触碰即重罚
严禁/限制投标条款、合同不利条款、保证金缺失等。触发即大幅扣分，严重者一票否决。
触碰红线 → 模块分归零或总分封顶
占比/目标类达标越近分越高
深耕区域占比、战略客户覆盖、EPC占比、业务结构偏差等。偏离战略目标越多分越低。
偏离 ≤ 5% → 100分 · 偏离 5-15% → 75分 · 偏离 > 15% → 40分
市场/校验类表现越好分越高
中标转化率、产值转化率、合同金额强度、数据完整度等。反映市场竞争力和数据可信度。
转化率 ≥ 60% → 100分 · 40-59% → 75分 · < 40% → 40分
